## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- 사용자의 닉네임
- 선호하는 음악 장르 2차원 리스트로 각각의 음악 정보(제목, 가수(그룹) 이름, 음악 장르 등) 리스트로 구성된 음악 데이터베이스를 만든다
- 사용자가 음악을 들으려는 목적

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 사용자가 입력한 선호 장르와 음악 데이터베이스의 장르를 비교하여 일치하는 곡에 기본 추천 점수를 부여한다.
- 사용자의 목적에 따라 음악의 속성 점수를 추출하여 가중치로 더하거나 곱해 최종 추천 점수를 계산한다.
- 계산된 최종 추천 점수를 기준으로 전체 음악 목록을 내림차순으로 정렬한다.

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 사용자의 입력 정보 확인 메시지
- 최종 추천 점수가 높은 순서대로 정렬된 음악 목록 TOP 3
